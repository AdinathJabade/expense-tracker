# Expense Tracker

A simple Python program to keep track of your daily expenses.

## How to use

1. Run the program:

   ```bash
   python main.py
2. Choose options to add new expenses or list all expenses.


# Features

Add expense with category, amount, date, and notes.

View all expenses.

Stores expenses in a JSON file.
